# 2️⃣ Simplified Magnet Score

We compute a **magnet score** for each price level.

Formula:

```
magnet_score = liquidation_size / distance_from_price
```

Where:

```
distance = abs(level - current_price)
```

Big liquidation + close distance = **strong magnet**.

---

# 3️⃣ Python Example

`liquidity_magnet.py`

```python
def compute_magnet_levels(current_price, liquidation_levels):

    magnets = []

    for level in liquidation_levels:

        price = level["price"]
        size = level["size"]

        distance = abs(price - current_price)

        if distance == 0:
            continue

        magnet_score = size / distance

        magnets.append({
            "price": price,
            "size": size,
            "score": magnet_score
        })

    magnets.sort(key=lambda x: x["score"], reverse=True)

    return magnets
```

# 6️⃣ Predicting Direction

We split levels:

```
below price → long liquidations
above price → short liquidations
```

If **below magnets > above magnets**:

➡ market likely **moves downward**

If **above magnets > below magnets**:

➡ market likely **moves upward**

---

# 7️⃣ Direction Prediction Script

```python
def predict_direction(current_price, magnets):

    below = 0
    above = 0

    for m in magnets:

        if m["price"] < current_price:
            below += m["score"]
        else:
            above += m["score"]

    if below > above:
        return "DOWNWARD LIQUIDITY TARGET"

    return "UPWARD LIQUIDITY TARGET"
```
